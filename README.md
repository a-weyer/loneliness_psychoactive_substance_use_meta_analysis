# Loneliness and Psychoactive Substance Use: Meta-Analysis

This repository contains the R code, processed data, tables, and figures for a meta-analysis examining the association between loneliness and psychoactive substance use.

The analysis was conducted as part of a term paper at the University of Rostock. The repository is intended to document the analytical workflow and support the reproducibility of the reported results.

## Research Objective

The aim of this meta-analysis is to examine the relationship between loneliness and psychoactive substance use.

The analyses include pooled effect estimates as well as subgroup analyses based on selected study characteristics.

### `IN_tables`

Contains input tables used during the analysis, including:

- `prisma_flow_data.csv`
- `raw_meta_table.csv`

### `code`

Contains the R scripts required to process the data, conduct the analyses, and generate the tables and figures.

- `clean_raw_data.R` Cleans and prepares the extracted study data.

- `analysis.R` Conducts the main meta-analytical models and subgroup analyses.

- `plots.R` Generates the forest plots and funnel plots.

- `tables.R` Creates the result tables included in the term paper.

- `prisma_flowchart.R` Generates the PRISMA flow diagram.

- `loneliness_substance_use.Rproj` RStudio project file for the analysis.

## Recommended Execution Order

The scripts should be executed in the following order:

1.  `code/clean_raw_data.R`
2.  `code/analysis.R`
3.  `code/plots.R`
4.  `code/tables.R`
5.  `code/prisma_flowchart.R`

Some scripts may depend on output files generated by earlier scripts.

## Data

The repository contains input tables, processed data objects, and the metadata table required for the analysis.

### Input Data

The folder `IN_tables/` contains the main input tables used by the R scripts.

### Processed Data

The folder `OUT_data/` contains processed R objects generated during the analytical workflow, including:

- `meta_k3.rds`
- `meta_k4.rds`
- `results_pool_k3.rds`
- `results_pool_k4.rds`

### Metadata

The folder `meta_table/` contains the extracted study information used for the meta-analysis.

Before reusing or redistributing the data, users should verify the terms of use and licensing conditions of the original data sources and publications.

## Results

The folder `results/` contains the figures and tables generated by the R scripts.

### Figures

The folder `results/figures/` includes:

- forest plots

### Tables

The folder `results/tables/` contains the main results tables in two formats:

- `docx/` for Microsoft Word documents

## Software Requirements

The analysis was conducted in R using RStudio.

To reproduce the analysis, users need:

- R
- the R packages loaded in the individual scripts

The required packages should be installed before running the scripts.

Example:

``` r
install.packages(c(
  "readxl",
  "tidyverse",
  "meta",
  "showtext",
  "grid",
  "magick",
  "flextable",
  "officer"
))
```

The exact list of required packages should be taken from the package-loading sections of the R scripts.

## Reproducing the Analysis

Clone the repository:

``` bash
git clone https://github.com/a-weyer/loneliness_psychoactive_substance_use_meta_analysis.git
```

Open the RStudio project:

``` text
code/loneliness_substance_use.Rproj
```

Then run the R scripts in the recommended order.

All file paths should be interpreted relative to the repository root or the RStudio project directory.

## Reproducibility Notes

- The repository does not include temporary R or RStudio session files.
- Automatically generated intermediate files are stored in `OUT_data/`.
- Tables and figures are stored in `results/`.
- Manually edited Inkscape files are excluded from the repository.
- Differences in R versions or package versions may lead to minor differences in the output.
