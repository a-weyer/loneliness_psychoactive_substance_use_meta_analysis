
# Loneliness, Social Isolation, and Subsequent Psychoactive Substance Use: A Systematic Review and Meta-Analysis of Longitudinal Studies

This repository contains the R code, processed data, tables, and figures underlying the meta-analysis.

The analysis was conducted as part of a term paper at the University of Rostock. The repository is intended to document the analytical workflow and support the reproducibility of the reported results.

## Research Objective

The aim of this systematic review and meta-analysis is to examine longitudinal associations of loneliness and social isolation with subsequent psychoactive substance use.

The analyses include pooled effect estimates as well as subgroup analyses based on selected study characteristics.

## Analytical Approach

Two parallel sets of meta-analyses were conducted:

`k4` refers to the primary analysis including four studies.
`k3` refers to the corresponding three-study analysis conducted after excluding one selected study.

The comparison between these analyses was used to examine how strongly the selected study influenced the pooled effect estimates.

The suffixes k3 and k4 are used consistently throughout the repository to distinguish the datasets, result objects, tables, and figures belonging to the respective analyses.

Further information on the review methodology, eligibility criteria, study-selection procedure, statistical models, and rationale for the exclusion of the selected study is provided in the associated term paper.

## Code

The `code/` folder contains the R scripts required to prepare the data, conduct the meta-analyses, and generate the result tables and figures.

| Script                            | Purpose                                                                                                |
  | --------------------------------- | ------------------------------------------------------------------------------------------------------ |
  | `01_clean_raw_meta_data.R`        | Cleans and prepares the extracted study-level data.                                                    |
  | `02_meta_analysis.R`              | Conducts the primary meta-analyses, the three-study analysis, and the corresponding subgroup analyses. |
  | `03_create_forest_funnel_plots.R` | Generates the forest plots, subgroup forest plots, and funnel plots.                                   |
  | `04_create_result_tables.R`       | Creates and exports the result tables included in the term paper.                                      |
  | `05_create_prisma_flowcharts.R`   | Generates the PRISMA flow diagrams.                                                                    |
  | `loneliness_substance_use.Rproj`  | RStudio project file for the analytical workflow.                                                      |

### Recommended Execution Order

The scripts should be executed in the following order:

1.  `01_clean_raw_meta_data.R`
2.  `02_meta_analysis.R`
3.  `03_create_forest_funnel_plots.R`
4.  `04_create_result_tables.R`
5.  `05_create_prisma_flowcharts.R`

## Data

The repository contains the original study-level extraction table, CSV input files used by the R scripts, and processed R objects generated during the analytical workflow.

### Study-Level Data Extraction

The `master_table/` folder contains:

- `meta_raw_extraction.xlsx`, the original Excel workbook used for study-level data extraction and effect-size coding

This workbook represents the original structured extraction table prepared for the systematic review.

### Input Data

The `input_data/` folder contains the files used directly as inputs by the R scripts:
  
- `raw_meta_table.csv`, a CSV export of the original Excel extraction table and the direct input for Script 01
- `prisma_flow_data.csv`, containing the counts and labels used to generate the PRISMA flow diagrams

Changes made to `master_table/meta_raw_extraction.xlsx` must be transferred to `input_data/raw_meta_table.csv` before the analytical workflow is rerun.

### Processed Data

The `processed_data/` folder contains processed R objects generated during the data-preparation and meta-analysis workflow:
  
- `meta_k4.rds`, containing the processed dataset used for the primary four-study analysis
- `meta_k3.rds`, containing the processed dataset used for the three-study analysis
- `results_pool_k4.rds`, containing the pooled results from the four-study analysis
- `results_pool_k3.rds`, containing the pooled results from the three-study analysis

## Reproducing the Analysis

1. Clone the repository:

```bash
git clone https://github.com/a-weyer/loneliness_psychoactive_substance_use_meta_analysis.git
```

2. Open the RStudio project:

```text
code/loneliness_substance_use.Rproj
```

3. Install any missing R packages listed under [Software Requirements](#software-requirements).
    
4. Run Scripts 01–04 in the recommended order.
    
5. Run Script 05 to recreate the automatically generated PRISMA flow diagrams.
    
6. Inspect the generated files in `processed_data/` and `results/`.
    
Because the RStudio project file is located in `code/`, the scripts access the other repository folders through relative paths.

## Reproducibility Notes

- The repository does not include temporary R or RStudio session files.
- The final workflow was tested by running all scripts in the recommended order.
- After their initial generation in R, the PRISMA flow diagrams were visually refined in Inkscape, for example by changing the font to Times New Roman.
- These manually edited Inkscape files are excluded from the repository. The automatically generated versions remain available in `results/figures/`.
- Differences in R or package versions may lead to minor differences in the generated output.
