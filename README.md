
# Loneliness, Social Isolation, and Subsequent Psychoactive Substance Use: A Systematic Review and Meta-Analysis of Longitudinal Studies

This repository contains the R code, processed data, tables, and figures underlying the meta-analysis.

The analysis was conducted as part of a term paper at the University of Rostock. The repository is intended to document the analytical workflow and support the reproducibility of the reported results.

## Research Objective

The aim of this systematic review and meta-analysis is to examine longitudinal associations of loneliness and social isolation with subsequent psychoactive substance use.

The analyses include pooled effect estimates as well as subgroup analyses based on selected study characteristics.

### `input_data`

Contains input data used during the analysis, including:

- `prisma_flow_data.csv`
- `raw_meta_table.csv`

### `code`

Contains the R scripts required to process the data, conduct the analyses, and generate the tables and figures.

- `01_clean_raw_meta_data.R` Cleans and prepares the extracted study data.

- `02_meta_analysis.R` Conducts the main meta-analytical models and subgroup analyses.

- `03_create_forest_funnel_plots.R` Generates the forest plots and funnel plots.

- `04_create_result_tables.R` Creates the result tables included in the term paper.

- `05_create_prisma_flowcharts.R` Generates the PRISMA flowcharts.

- `loneliness_substance_use.Rproj` RStudio project file for the analysis.

## Recommended Execution Order

The scripts should be executed in the following order:

1.  `code/01_clean_raw_meta_data.R`
2.  `code/02_meta_analysis.R`
3.  `code/03_create_forest_funnel_plots.R`
4.  `code/04_create_result_tables.R`
5.  `code/05_create_prisma_flowcharts.R`

Some scripts may depend on output files generated by earlier scripts.

## Software Requirements

The analysis was conducted in R using RStudio.

To reproduce the analysis, users need:

- R
- the R packages loaded in the individual scripts

The required packages should be installed before running the scripts.

Example:

``` r
install.packages(c(
  "readxl",
  "tidyverse",
  "meta",
  "showtext",
  "magick",
  "flextable",
  "officer"
))
```

The exact list of required packages should be taken from the package-loading sections of the R scripts.

## Reproducing the Analysis

Clone the repository:

``` bash
git clone https://github.com/a-weyer/loneliness_psychoactive_substance_use_meta_analysis.git
```

Open the RStudio project:

``` text
code/loneliness_substance_use.Rproj
```

Then run the R scripts in the recommended order.

All file paths should be interpreted relative to the repository root or the RStudio project directory.

## Reproducibility Notes

- The repository does not include temporary R or RStudio session files.
- The final workflow was tested by running all scripts in the recommended order.
- Manually edited Inkscape files are excluded from the repository.
- Differences in R or package versions may lead to minor differences in the generated output.
